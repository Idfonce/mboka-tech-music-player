# MBOKA-TECH Music Player 🎵

A beautiful offline music player that works entirely in your browser. Play your local music files without any internet connection!

## Features ✨

- 🎵 Play local MP3, WAV, and audio files
- ❤️ Favourite songs collection
- 📋 Create and manage playlists
- 🔍 Search your music library
- ⏱️ Recently added view
- 👤 Profile with custom picture
- 📊 Storage usage statistics
- 🌈 Beautiful gradient design
- 📱 Mobile responsive
- 💾 All data stored locally

## How to Use 📱

1. Open the app
2. Grant file access permission
3. Click "Choose Files" to select your music
4. Start playing!

## Live Demo 🌐

Visit: https://mboka-tech.github.io/music-player

## Installation on Phone 📲

### As PWA
1. Open in Chrome
2. Click "Add to Home Screen"
3. Use like a native app!

### As APK
1. Download from releases
2. Install and grant storage permission
3. Enjoy!

## Technologies 💻

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage API
- Web Audio API
- Service Workers
